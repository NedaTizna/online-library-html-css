Project Description

This website is a simple online library created as a sample project for the field of Library and Information Science.
The goal is to demonstrate a basic skills in HTML and CSS by building a small, user-friendly informational site.

The website includes a homepage, a list of sample books,and this "about" page. It was developed as part of a personal portfolio to showcase an interest in organizing informaion and creating simple digital platforms.

No backend or JavaScript is used in this version-the project is entirely built with HTML and CSS.
